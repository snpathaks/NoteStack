# docker-notes-api

A tiny REST API to store and retrieve notes, built with Node.js and Redis, fully containerized using Docker and Docker Compose.

---



## 🚀 What This Project Does

- Add notes via an HTTP API
- Retrieve all saved notes
- Store data in Redis
- Run everything using Docker containers

This project is designed to help beginners understand Docker fundamentals with a very small codebase.

---

## 🧰 Tech Stack

- Node.js
- Express
- Redis
- Docker
- Docker Compose

---

## ⚙️ Prerequisites

Make sure you have installed:

- Docker
- Docker Compose (comes with Docker Desktop)

---

From the root folder (`notes-docker`):

```bash::
docker compose up --build
